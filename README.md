# English Quest — Учи английский играючи

Интерактивное SPA для изучения английского языка через игры. Next.js 14, TypeScript, Tailwind CSS, shadcn/ui, framer-motion, Zustand.

## Запуск

```bash
npm install
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000).

Или дважды кликнуть `start.bat` — откроет браузер и запустит сервер.

## Игры (10 шт.)

| # | Игра | Описание |
|---|------|----------|
| 1 | СловоХранилище | Карточки слов с переводом, транскрипцией и примерами |
| 2 | ГраммоКузница | Заполни пропуск правильной формой + объяснение правила |
| 3 | Эхо-Камера | Слушай слово и записывай услышанное (Web Speech API) |
| 4 | СпеллШторм | Напиши английское слово по русскому переводу |
| 5 | ЛабИдиом | Угадай значение идиомы + история происхождения |
| 6 | СловоЛов | Найди слова в сетке букв за 90 секунд |
| 7 | ВремПортал | Впиши глагол в нужном времени |
| 8 | ФразоБум | Собери падающие слова в правильное предложение |
| 9 | СинонимБой | Дуэль синонимов против AI-соперника |
| 10 | ДиалогМастер | Заполни пропуски в диалогах, прослушай целиком |

## Мой словарь

2554 самых частотных английских слов с переводом. Отмечай выученные, отслеживай прогресс.

## Цифры и Цвета

Два режима: угадывай русские названия цветов и переводи числа из слов в цифры.

## Особенности

- Тёмная/светлая тема
- Всё сохраняется в localStorage
- Работает без интернета
- Анимированные переходы, конфетти, shake-эффекты
- Система очков и ачивок
