# English Quest — Учи английский играючи

Интерактивное SPA для изучения английского языка через 13 игр, словарь на 2500+ слов и тренировку цветов и чисел. Next.js 14, TypeScript, Tailwind CSS, shadcn/ui, framer-motion, Zustand, lucide-react.

## Запуск

```bash
npm install
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000).

Или дважды кликнуть `start.bat` — автоматически установит зависимости, запустит сервер и откроет браузер.

## Игры (13 шт.)

| # | Игра | Механика |
|---|------|----------|
| 1 | **СловоХранилище** | Карточки слов с переводом, транскрипцией, определением, синонимами и примерами. Коллекция выученных слов |
| 2 | **ГраммоКузница** | Заполни пропуск правильной грамматической формой. Категории: времена, предлоги, артикли, условные предложения, модальные глаголы. Объяснение после каждого ответа |
| 3 | **Эхо-Камера** | Слушай слово/фразу через Web Speech API и записывай услышанное. Подсказки: первая буква, длина. Регулируемая скорость озвучки |
| 4 | **СпеллШторм** | Напиши английское слово по русскому переводу. 3 попытки, подсветка правильных/неправильных букв (Wordle-стиль). Режим на скорость: 60 секунд |
| 5 | **ЛабИдиом** | Угадай значение идиомы из 4 вариантов. История происхождения, пример в диалоге. Категории: бизнес, повседневные, фразовые глаголы, пословицы |
| 6 | **СловоЛов** | Найди как можно больше слов в сетке букв за 90 секунд. Сетка 4×4, слова от 4 букв. Озвучка найденных слов, список пропущенных в конце |
| 7 | **ВремПортал** | Напиши правильную форму глагола в указанном времени. Визуальная шкала времени (прошлое → настоящее → будущее), таблица спряжения при ошибке |
| 8 | **ФразоБум** | Падающие слова — собери их в правильном порядке, чтобы получилось осмысленное предложение. Бонус за скорость |
| 9 | **СинонимБой** | Дуэль синонимов против AI-соперника. Называй синонимы по очереди, у кого первого закончатся — проиграл. Банк из 100+ слов с 3-6 синонимами |
| 10 | **ДиалогМастер** | Восстанови пропущенные реплики в диалогах. 20+ сценариев: ресторан, аэропорт, собеседование, врач. Озвучка всего диалога целиком |
| 11 | **Словесная Лестница** | Меняй по одной букве, чтобы дойти от первого слова до последнего. Все промежуточные слова должны быть настоящими. 15 предопределённых головоломок |
| 12 | **Найди Пару** | Переворачивай карточки и ищи пары: английское слово + русский перевод. 16 карточек (8 пар), 3 раунда. Анимация переворота |
| 13 | **Анаграммы** | Буквы перепутаны — расставь правильно. Русский перевод как подсказка. Таймер 25 секунд на слово, бонус за скорость |

## Цифры и Цвета

Два режима в одной игре:
- **Цвета** — показывается английское название цвета, нужно написать русский перевод (42 цвета)
- **Цифры** — число написано прописью на английском, нужно ввести цифрами (1–1000)

## Мой словарь

- 2500+ самых частотных английских слов с переводом (A1-B2)
- Вкладка «Изучение» — карточки по 10 слов, кнопка «Выучил» для мгновенного переноса в выученные
- Вкладка «Мои слова» — все выученные с поиском и пагинацией (по 30 слов)
- Прогресс-бар, кнопка сброса с подтверждением
- Всё хранится в localStorage

## Технологии

- **Next.js 14** (App Router, все страницы статические)
- **TypeScript** — строгая типизация
- **Tailwind CSS** — кастомная тема, тёмный/светлый режим через class-стратегию
- **shadcn/ui** — Button, Card, Badge, Dialog, Tabs, Select, Slider, Progress, Tooltip, Switch, ScrollArea, Input
- **framer-motion** — анимации появления, переходов, правильных/неправильных ответов, конфетти, плавающие буквы
- **lucide-react** — все иконки
- **Zustand** — управление состоянием, персистентность в localStorage
- **Web Speech API** — озвучка слов в играх на аудирование

## Особенности

- Полностью на русском языке
- Тёмная/светлая тема (переключатель в правом верхнем углу)
- Кнопка перезапуска приложения (левый верхний угол)
- Всё сохраняется в localStorage — прогресс не теряется при перезагрузке
- Работает полностью офлайн — только фронтенд, без бэкенда
- Адаптивный дизайн (mobile-first), стеклянный морфизм (glassmorphism)
- Система очков, streak-бонусов и ачивок (18 шт.)
- Анимации: confetti, shake, scale, fade, плавающие фигуры на фоне
